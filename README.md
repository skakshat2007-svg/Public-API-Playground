# 🚀 Public API Playground

A modern, responsive React application that showcases the power of public APIs. Explore random dog images, jokes, and user profiles with a sleek dark-mode interface.

## ✨ Features

- **🐶 Dog Finder**: Fetch random dog images and see their breed names. Includes a "Copy Image URL" button for easy sharing.
- **😂 Joke Generator**: Get random jokes with a setup and punchline. Interactive "Next Joke" flow.
- **👤 Random User Profile**: Generate detailed random user profiles including name, avatar, email, location, age, and phone number.

## 🛠️ Tech Stack

- **React**: Functional components and Hooks (`useState`, `useEffect`).
- **Vite**: Ultra-fast frontend build tool.
- **Vanilla CSS**: Clean, modular styling with glassmorphism and smooth animations.
- **Public APIs**:
  - [Dog CEO API](https://dog.ceo/dog-api/)
  - [Official Joke API](https://official-joke-api.appspot.com/random_joke)
  - [Random User Generator API](https://randomuser.me/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/skakshat2007-svg/Public-API-Playground.git
   cd Public-API-Playground
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` folder, ready for deployment on Netlify or Vercel.
